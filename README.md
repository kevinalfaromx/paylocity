# `Paylocity Dashboard Playwright Tests UI & API`

## `Getting Started`

Clone the repo
```sh
git clone https://github.com/kevinalfaromx/paylocity.git
```
Use the latest node version:
```sh
nvm install OR download from https://nodejs.org/en/download/
```
Execute the next command to install dependencies:
```sh
npm install
```
Execute playwright tests win the next command: 
```sh
npx playwright test tests
```
